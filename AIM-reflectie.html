Reflectie

1) AIM-reflectie op WPL1

In werkplekleren leren we praktisch toepassen wat we in de andere cursussen hebben geleerd. Ook leren we meer over onszelf en hoe we ons kunnen oriënteren in ons traject.

Ik vond het leuk om applicaties te bouwen die een mooi eindresultaat hadden. Zo hebben we verschillende applicaties geschreven in c# tijdens de PE's en labo's. Werken met git en git bash was initieel moeilijk, maar het is leuk om een plek te hebben waar we onze projecten kunnen verzamelen. Git is zeer praktisch en fijn om mee te werken.

Het is frustrerend dat we voor de PE's redelijk weinig tijd krijgen. Het is altijd een grote uitdaging om alles binnen de gegeven tijd af te werken. Dit zorgt voor bijkomende stress, wat niet altijd positief is. Tijdens de PE van HTML/CSS had ik hier veel last van waardoor ik niet optimaal heb kunnen werken. Het viel op dat ik na het schrijven ven de HTML niet meer veel tijd over had, waardoor ik te gestresseerd was om mijn CSS-opmaak te verzorgen.

Wat ik heb geleerd is dat ik goed voorbereid moet zijn op elke PE. Nu weet ik wat ik kan verwachten zodat ik niet overweldigd word. Mijn tempo in het coderen is redelijk goed, maar ik moet hier meer aan werken. Ik heb al mijn opdrachten en PE's goed kunnen volbrengen, waardoor ik ervan overtuigd ben dat deze opleiding goed bij mij past.

Ik ben trots op het feit dat ik goeie resultaten heb behaald op mijn PE's en opdrachten. Ik twijfelde over mijn leervermogen, maar uit mijn resultaten blijkt dat dit geen probleem is.
Kernkwaliteit: analyseren
Valkuil: te kritisch zijn, overanalyseren
Allergie: nonchalance
Uitdaging: genoegen nemen met een voorlopige analyse
  • Welke talenten/kwaliteiten bezit ik vandaag en heb ik kunnen toepassen tijdens Werkplekleren? Welke talenten zou ik nog meer kunnen inzetten tijdens Werkplekleren?

    Tijdens Werkplekleren heb ik verschillende kwaliteiten kunnen gebruiken, zoals doorzettingsvermogen, inzet, kritisch denken... Wat ik nog meer kan inzetten is zelfverzekerdheid en efficiëntie.
  • Hoe zit het met mijn valkuilen? Wat gaat nog niet zo goed en waarom?

    Een valkuil van te veel inzet is overbelasting. Door het semester heen hebben we regelmatig opdrachten en PE's. Als we veel te doen hebben in een week kan het zijn dat er weinig tijd overblijft voor ontspanning. Hier moet ik een goeie balans in vinden.
  • Wat zijn verbeterpunten of uitdagingen voor mezelf? Wat kan ik concreet doen om dit aan te pakken? Welke hulp zou welkom zijn? Wat kan ik vragen aan mijn lectoren / mijn werkplekcoach?

    Ik wil verder werken aan mijn efficiëntie. In de kerstvakantie ga ik hiervoor een cursus volgen om sneller te coderen. Ook ga ik oefenen op blind typen. Verder weet ik niet waar ik nog hulp bij nodig heb, maar alle hulp is welkom.
  • Wat wil ik nog meer bijleren tijdens de volgende periode van WPL? Op welk vlak wil en kan ik mezelf nog verder ontwikkelen?

    In Werkplekleren 2 wil ik oefenen op communicatie en samenwerking. Op een duidelijke en correcte manier communiceren kan soms moeilijk zijn. Dit is toch belangrijk voor een goeie samenwerking te bekomen.

3) X-factor

Het is mijn passie positieve zaken op te bouwen. Zo heb ik tijdens de PE's van c# verschillende applicaties gebouwd met een mooi eindresultaat. Dit geeft mij voldoening.

Bij de opdrachten rond HTML en CSS ben ik op zoek gegaan methoden om effecten toe te voegen die buiten de scope van het curriculum liggen. Ik heb het heft in eigen handen genomen om resultaten te verwezenlijken waar ik trots op kan zijn.

Tijdens de lessen van IT-organisation heeft de lector ons verschillende opdrachten gegeven waar we samen moesten werken. We moesten samen antwoorden vinden op de vragen. Zo oefenden we onze communicatie en ons luisterend vermogen.

De vakken c# essentials, web essentials en data essentials zijn essentieel om onze disciplinaire kennis te vergroten. In werkplekleren leren we deze vaardigheden praktisch toepassen. Door de oefeningen op te lossen die in de cursus staan, leren we onze kennis toe te passen. Tijdens de PE's wordt onze disciplinaire kennis getest.
